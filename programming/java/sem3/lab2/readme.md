# Лабораторная работа 2
«Работа со строками»

* При работе со строками использовать классы **StringBuffer** или **StringBuilder**. Если нужно разбивать строку на элементы, использовать класс **StringTokenizer**.
* Строка представляет собой арифметическое выражение, состоящее из целых констант, *+, -, *, /, (, )*. 
* Представить выражение в обратной польской нотации и вычислить её значение.
